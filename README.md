# 🛡️ VeriScan | The AI-Powered Digital Authenticity Suite

VeriScan is a comprehensive, enterprise-grade cybersecurity web application designed to detect digital fraud, synthetic media, and social engineering threats. Built with a sleek, dark-themed "glassmorphism" interface, it utilizes advanced computer vision, audio analysis, and Google's Multimodal Large Language Models to authenticate digital media in real-time.

---

## ✨ Key Features

1. **🔗 PhishGuard (URL & Sandbox Analysis)**
   - Safely captures a high-resolution screenshot of any suspicious URL in an isolated sandbox.
   - Uses AI vision to cross-reference brand logos against the actual domain to detect typosquatting and credential-harvesting forms.

2. **🎥 DeepFake Sentry (Video Forensics)**
   - Extracts frames and audio from uploaded video files (`.mp4`, `.avi`).
   - Analyzes blink patterns (Eye Aspect Ratio), facial liveness, and synthetic audio frequencies to detect deepfakes generated by tools like Wav2Lip or Viggle.

3. **🖼️ AI Image Check (Synthetic Image Detection)**
   - Scans images (`.jpg`, `.png`) for anatomical anomalies, unnatural textures, and physically impossible lighting.
   - Accurately flags content generated by Midjourney, DALL-E, or Stable Diffusion.

4. **📄 DocuGuard (Invoice & ID Forgery Check)**
   - Performs Optical Character Recognition (OCR) and visual forensics on documents.
   - Detects mismatched fonts, illogical math, and JPEG compression "halos" indicative of digital manipulation and fraud.

## 💻 Technology Stack

* **Backend:** Python 3.12, Flask, Gunicorn
* **Frontend:** HTML5, CSS3, Bootstrap 5, Jinja2, Uiverse UI
* **AI & Machine Learning:** Google Gemini 2.5 Flash API (`google-genai`)
* **Computer Vision & Audio:** OpenCV (`opencv-python-headless`), MediaPipe, Librosa, NumPy, Pillow
* **External APIs:** Screenshot API (screenshotapi.net)
* **Deployment:** Render (Cloud PaaS)

## 🚀 Local Installation & Setup

Follow these steps to run VeriScan on your local machine.

## 1. Clone the Repository

- git clone https://github.com/YourUsername/VeriScan.git
- cd VeriScan


## 2. Create a Virtual Environment


### On Windows:
- python -m venv venv

- venv\Scripts\activate

### On Mac/Linux:

- source venv/bin/activate


## 3. Install Dependencies

- pip install -r requirements.txt


## 4. Set Up Environment Variables

- Create a file named .env in the root directory and add your API keys:


- **GEMINI_API_KEY**=your_gemini_api_key_here
- **SCREENSHOT_API_KEY**=your_screenshot_api_key_here


## 5. Run the Application

- python app.py


**The app will now be running at http://127.0.0.1:5000.**
